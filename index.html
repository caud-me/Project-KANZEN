Latest Update v0.9.1-alpha

Implemented a live update system: new beatmaps can now be downloaded automatically without updating the entire game app. by whyeron
The game now validates local files and skips the connection check if played offline (after the initial setup), new players are still required to connect to the internet. by whyeron
Fixed an issue where you can press back and go to the home screen while still downloading game files during initial setup, this results to library page with no beatmaps installed. by whyeron
New changelog website. Not a final website!! i just wanted to see my ideas come to life then improve it in the future. by whyeron

Previous Updates

Refined game user interface, kanzen's visual identity revolves around red-pink and gray aesthetic. by whyeron
Beatmaps are no longer stored inside the app, instead all downloaded beatmaps will be saved to your device data folder Android/data for android. AppData for windows. by whyeron
Lanes and touch controls are now responsive, they are centered regardless of viewport width. by whyeron
Introduced song lyrics for beatmaps, they appear at the bottom of the in game screen, available when a beatmap has a LRC file or just .txt with LRC format. by whyeron
Dynamic backgrounds and audio, the game will randomly pick a beatmap and play it as the menu music and background (upon opening the game) then uses the song/background from the last beatmap you played. by whyeron
New particle effects when hitting a note with lane flash improvements. by whyeron
Minor beatmap parser enhancements, planning to make a parser that supports kanzen files and osu mania files. by whyeron
